<h1 style="text-align: center;"><strong><span style="color: rgb(0, 0, 0);">Definici&oacute;n de proyecto:</span></strong></h1>
<p style="text-align: center;"><br></p>
<p>Se pide crear una aplicación que permita la traducción de textos simples de un idioma a otro,
seleccionando idioma de origen e idioma de destino y mostrando su respectiva traducción.</p>

  <ol>
    <li>Idioma Origen:
      Utilizando un elemento de lista, esta opción le debe permitir
seleccionar de que idioma se realizará la traducción.</li>
    <li>Idioma Destino: Utilizando un elemento de lista, esta opción le debe permitir
seleccionar a que idioma se realizará la traducción.</li>
    <li>Traducir: Este botón le permitirá traducir el texto escrito en el cuadro de texto 1 en el
cuadro de texto 2.</li>
    <li>Color Letra: Este botón le permitirá cambiar el color de la letra del cuadro de texto 1 o el cuadro de texto 2.</li>
    <li>Tamaño Letra: Este botón le permitirá cambiar el tamaño de la letra del cuadro de
texto seleccionado.</li>
  </ol>
<p>Nota: El traductor deberá tener la capacidad de traducir como mínimo 50 palabras y a 3
idiomas.</p>
